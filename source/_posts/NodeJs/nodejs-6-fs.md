# 是什么？
fs是node内置核心模块，提供文件系统操作的API。

**Fs模块-基本操作**
  - start
  - write
  - read
  - watcher

**Fs模块-常用方法**
  - 权限操作
    - chmod
    - chgrp
  - 文件增删改查
    - 打开/关闭
    - 读取
    - 写入
    - 删除
  - 工具方法
